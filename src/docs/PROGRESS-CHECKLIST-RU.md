## 🐛 Решение проблем

### Проблема: Проценты = 0 везде
- [ ] Проверил что формула в правильных ячейках (C9, E9, G9...)
- [ ] Проверил что формула **не пустая**
- [ ] Проверил что камеры **активны** (есть циклы "In Progress")

### Проблема: Проценты неправильные (2, 3, 1 вместо 20, 30, 10)
- [ ] Скопировал **ОБНОВЛЁННЫЙ** код из `/docs/google-apps-script-progress.txt`
- [ ] Убедился что код умножает на 100: `percentValue = percentValue * 100`
- [ ] Запустил скрипт заново

### Проблема: Ошибка "urlfetch limit"
- [ ] Открыл **Triggers** → удалил ВСЕ триггеры
- [ ] Создал **ОДИН НОВЫЙ** триггер (каждые 15 минут)
- [ ] Подождал до завтра (лимиты сбросятся)
- [ ] Прочитал `/docs/GOOGLE-SHEETS-URLFETCH-LIMIT-FIX.md`

### Проблема: Уведомления не работают
- [ ] Разрешил уведомления в браузере (popup запроса)
- [ ] Проверил что камера в зоне 99-103%
- [ ] Проверил что фото **не загружено**
- [ ] Подождал **2 минуты** (первое уведомление не сразу)
- [ ] Проверил звук в браузере (не отключён)

### Проблема: Процент >100% но круг пустой
- [ ] Это **нормально**! Для >100% круг показывает остаток (modulo 100)
- [ ] Пример: 115.5% → круг заполнен на 15.5%
- [ ] Цвет должен быть **фиолетовый** 🟣
