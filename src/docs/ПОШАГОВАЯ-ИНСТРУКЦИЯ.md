# 📋 Пошаговая инструкция настройки (всё подробно)

## Важно понять схему работы:

```
┌─────────────────────────────────────────────────────────────────┐
│  ВАШ СКРИПТ                                                     │
│  (заполняет Google Таблицу автоматически)                       │
└────────────────┬────────────────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  GOOGLE SHEETS                                                  │
│  Таблица с 6 столбцами                                          │
│  5 | 0123 | 0124 | Beržas235 | 2024-02-23 16:00 | 16:05        │
└────────────────┬────────────────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  GOOGLE APPS SCRIPT (МОЙ КОД)                                   │
│  Автоматически обнаруживает новую строку →                      │
│  Отправляет данные на ваш сервер                                │
└────────────────┬────────────────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  ВАШ SUPABASE СЕРВЕР                                            │
│  1. Закрывает старый цикл 0123                                  │
│  2. Ждёт 2-3 минуты                                             │
│  3. Создаёт новый цикл 0124                                     │
└────────────────┬────────────────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  ПРИЛОЖЕНИЕ                                                     │
│  Оператор видит новый цикл                                      │
└─────────────────────────────────────────────────────────────────┘
```

# ЧАСТЬ 1: Подготовка учётных данных (5 минут)

## Шаг 1.1: Получите данные из Supabase

1. Откройте ваш проект в Supabase: https://supabase.com/dashboard

2. Найдите **Project ID**:
   - Посмотрите на URL в браузере
   - Он выглядит так: `https://supabase.com/dashboard/project/[ЭТО_ВАШ_ID]`
   - Скопируйте эту часть после `/project/`
   - **Сохраните куда-нибудь**, например:
     ```
     Project ID: abcdefghijklmnop
     ```

3. Найдите **SUPABASE_ANON_KEY**:
   - В левом меню нажмите иконку шестерёнки (⚙️) внизу
   - Выберите **API**
   - Найдите раздел "Project API keys"
   - Скопируйте значение поля **anon public**
   - **Сохраните куда-нибудь**:
     ```
     Anon Key: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
     ```

**✅ Готово!** Теперь у вас есть 2 важные строки.

# ЧАСТЬ 2: Создание Google Таблицы (3 минуты)

## Шаг 2.1: Создайте новую таблицу

1. Откройте Google Sheets: https://sheets.google.com

2. Нажмите **+ Создать** (или Blank / Пустой документ)

3. Назовите таблицу, например: **"Циклы сушки - автосинхронизация"**

## Шаг 2.2: Добавьте заголовки столбцов

В **первой строке** введите заголовки (это для вас, чтобы понимать что где):

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| Камера | Старый номер | Новый номер | Порода | Конец старого | Начало нового |

**Или по-литовски (как вам удобнее):**

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| Kamera | Senas nr | Naujas nr | Rūšis | Seno pabaiga | Naujo pradžia |

⚠️ **Важно:** Заголовки нужны только для вас, скрипт работает по номерам столбцов (A, B, C...), а не по названиям.

## Шаг 2.3: Запомните название листа

1. Внизу таблицы есть вкладка с названием листа (обычно "Лист1" или "Sheet1")
2. Если хотите, можете переименовать (щелчок правой кнопкой → Переименовать)
3. **Запомните это название**, например: `Лист1`

**✅ Готово!** Таблица создана.

# ЧАСТЬ 3: Настройка Google Apps Script (10 минут)

## Шаг 3.1: Откройте редактор скриптов

1. В вашей Google Таблице откройте меню **Расширения** (Extensions)

2. Выберите **Apps Script**

3. Откроется новая вкладка с редактором кода

4. Удалите весь существующий код (там будет что-то типа `function myFunction() {}`)

## Шаг 3.2: Вставьте код

1. Откройте файл в документации: `/docs/google-apps-script.gs`

2. Скопируйте **ВЕСЬ КОД** (от начала до конца, около 600 строк)

3. Вставьте в редактор Apps Script

4. **Сохраните**: Ctrl+S (Windows/Linux) или Cmd+S (Mac)

## Шаг 3.3: Настройте константы

Найдите в начале кода (строки 15-30) эти строки:

```javascript
const URL_WEBHOOK = 'https://[ваш-project-id].supabase.co/functions/v1/make-server-c5bcdb1f/sheets/process-row';
const AUTH_TOKEN = 'ваш_anon_key_здесь';
const SHEET_NAME = 'Лист1';
```

**Измените их:**

### 3.3.1: URL_WEBHOOK

Замените `[ваш-project-id]` на ваш Project ID из Шага 1.1:

```javascript
const URL_WEBHOOK = 'https://abcdefghijklmnop.supabase.co/functions/v1/make-server-c5bcdb1f/sheets/process-row';
```

### 3.3.2: AUTH_TOKEN

Вставьте ваш Anon Key из Шага 1.1:

```javascript
const AUTH_TOKEN = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...вся_длинная_строка...';
```

### 3.3.3: SHEET_NAME

Укажите название вашего листа из Шага 2.3:

```javascript
const SHEET_NAME = 'Лист1';  // или как вы назвали
```

## Шаг 3.4: Проверьте номера столбцов

Найдите в коде (строки 22-29):

```javascript
const COLUMNS = {
  CHAMBER_NUMBER: 1,        // Столбец A
  OLD_SEQUENTIAL: 2,        // Столбец B
  NEW_SEQUENTIAL: 3,        // Столбец C
  WOOD_TYPE_LITHUANIAN: 4,  // Столбец D
  OLD_END_DATE: 5,          // Столбец E
  NEW_START_DATE: 6         // Столбец F
};
```

⚠️ **Если ваш скрипт будет заполнять данные в ДРУГИХ столбцах**, измените номера здесь.

**Например**, если ваш скрипт пишет:
- Номер камеры в столбец C (3-й столбец)
- Старый номер в столбец D (4-й столбец)
- и т.д.

То измените на:
```javascript
const COLUMNS = {
  CHAMBER_NUMBER: 3,        // Столбец C
  OLD_SEQUENTIAL: 4,        // Столбец D
  ...
};
```

## Шаг 3.5: Сохраните

Нажмите **Сохранить** (иконка дискеты или Ctrl+S)

**✅ Готово!** Код настроен.

# ЧАСТЬ 5: Автоматизация (2 минуты)

### Вариант А: Реагировать при изменении таблицы (рекомендуется)

**Плюсы:** Мгновенная реакция на новую строку  
**Минусы:** Срабатывает при любом изменении таблицы

1. В Apps Script выберите функцию **setupTrigger**

2. Нажмите **Выполнить** (▶️)

3. В логах увидите: **"✅ Триггер onEdit создан успешно"**

4. Готово! Теперь при добавлении новой строки скрипт сработает автоматически.

### Вариант Б: Проверять каждые 5 минут

**Плюсы:** Не мешает редактированию таблицы  
**Минусы:** Задержка до 5 минут

2. Нажмите **Выполнить** (▶️)

3. В логах увидите: **"✅ Периодический триггер (каждые 5 минут) создан успешно"**

   - **Вариант А:** `onEdit` - при изменении таблицы

# ЧАСТЬ 6: Настройка вашего скрипта

Теперь настроим **ваш** скрипт, который будет заполнять таблицу.

## Формат данных для вашего скрипта

Когда ваш скрипт добавляет новую строку в Google Таблицу, он должен заполнить **6 столбцов**:

### Столбец A: Номер камеры
- **Тип:** Целое число
- **Диапазон:** 1-21
- **Пример:** `5`

### Столбец B: Старый номер цикла
- **Тип:** Текст (4 цифры)
- **Формат:** `0000` - `9999`
- **Пример:** `0123`, `0045`, `1234`
- **Важно:** Обязательно 4 цифры! Если номер 1, пишите `0001`

### Столбец C: Новый номер цикла
- **Тип:** Текст (4 цифры)
- **Формат:** `0000` - `9999`
- **Пример:** `0124`, `0046`, `1235`
- **Важно:** Обязательно 4 цифры!

### Столбец D: Литовское название породы
- **Тип:** Текст
- **Допустимые значения:**
  - `Beržas235` (или `Berzas235`) → Берёза 235
  - `Beržas285` (или `Berzas285`) → Берёза 285
  - `Alksnis235` → Ольха 235
  - `Ąžuolas235` (или `Azuolas235`) → Дуб 235
  - `Uosis235` → Ясень 235
  - `Klevas235` → Клён 235
  - `Skroblas235` → Скробла 235

**⚠️ Важно:** Используйте **ТОЧНО** эти названия, включая число 235/285!

### Столбец E: Дата завершения старого цикла
- **Тип:** Дата и время
- **Формат:** `ГГГГ-ММ-ДД ЧЧ:ММ:СС` или объект Date
- **Примеры:**
  - `2024-02-23 16:00:00`
  - `2024-12-31 23:59:59`
  - `new Date()` (если используете JavaScript)

### Столбец F: Дата начала нового цикла
- **Тип:** Дата и время
- **Формат:** `ГГГГ-ММ-ДД ЧЧ:ММ:СС` или объект Date
- **Примеры:**
  - `2024-02-23 16:05:00`
  - Обычно на 5-10 минут позже столбца E

## Пример строки для вашего скрипта

```javascript
// Если ваш скрипт на JavaScript (Google Apps Script)
const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Лист1');

sheet.appendRow([
  5,                          // A: Номер камеры
  '0123',                     // B: Старый номер (обязательно строка!)
  '0124',                     // C: Новый номер
  'Beržas235',               // D: Порода
  '2024-02-23 16:00:00',     // E: Конец старого
  '2024-02-23 16:05:00'      // F: Начало нового
]);
```

```python
# Если ваш скрипт на Python (gspread)
worksheet.append_row([
    5,                          # A: Номер камеры
    '0123',                     # B: Старый номер
    '0124',                     # C: Новый номер
    'Beržas235',               # D: Порода
    '2024-02-23 16:00:00',     # E: Конец старого
    '2024-02-23 16:05:00'      # F: Начало нового
])
```

## Важные моменты для вашего скрипта

### ✅ Обязательно:
- Все 6 столбцов должны быть заполнены
- Номера циклов - 4 цифры (с нулями впереди: `0001`, `0012`, `0123`)
- Название породы - точно как в списке выше
- Номер камеры - от 1 до 21

### ❌ Не нужно:
- Не создавайте циклы в приложении вручную
- Не заполняйте столбец с ID (его нет в таблице)
- Не добавляйте дополнительные столбцы (кроме этих 6)

# ЧАСТЬ 9: Устранение проблем

## Проблема: "Лист не найден"

**Решение:**
- Проверьте `SHEET_NAME` в коде Apps Script
- Название должно **точно** совпадать с названием вкладки внизу таблицы
- Учитывается регистр! "Лист1" ≠ "лист1"

## Проблема: "Ошибка HTTP 401" или "Unauthorized"

**Решение:**
- Проверьте `AUTH_TOKEN` - должен быть **anon public** ключ из Supabase
- Скопируйте ключ заново, убедитесь что нет лишних пробелов

## Проблема: "Ошибка HTTP 404"

**Решение:**
- Проверьте `URL_WEBHOOK`
- Project ID должен быть правильным
- URL должен заканчиваться на `/sheets/process-row`

## Проблема: Старый цикл не закрывается

**Возможные причины:**
- Номер в столбце B не совпадает с `sequentialNumber` в базе
- Номер камеры не совпадает

**Решение:**
- Откройте приложение → найдите цикл
- Посмотрите его "Порядковый номер" (sequentialNumber)
- В столбце B должен быть **ТОЧНО** такой же номер (с нулями!)
- Пример: если в приложении `123`, в таблице должно быть `0123`

## Проблема: Новый цикл не создаётся

**Возможные причины:**
- Литовское название породы не распознано
- Название написано с ошибкой

**Решение:**
- Используйте **ТОЧНЫЕ** названия из списка в Части 6
- Проверьте регистр букв (заглавные!)
- Проверьте число в конце (235 или 285)

## Проблема: Триггер не срабатывает

**Решение:**
1. Apps Script → левое меню → иконка будильника (Triggers)
2. Проверьте что триггер есть в списке
3. Если нет - запустите `setupTrigger()` или `setupTimeTrigger()` снова

## Проблема: Появляются дубликаты циклов

**Причина:** Строка обработалась несколько раз

**Решение:**
- Система автоматически предотвращает дублирование
- Если всё равно появляются - проверьте что не запущено несколько триггеров
- Apps Script → Triggers → удалите лишние триггеры

# ЧАСТЬ 10: Полезные команды

## Ручная синхронизация одной строки

В Apps Script запустите функцию:
```javascript
manualSync()
```

Обработает последнюю строку в таблице.

## Синхронизация всех строк

В Apps Script запустите функцию:
```javascript
syncAllRows()
```

⚠️ Осторожно! Обработает ВСЕ строки в таблице (кроме заголовка).

## Проверка конфигурации

В Apps Script запустите функцию:
```javascript
testConfiguration()
```

Покажет все настройки и проверит доступ к таблице.

## Удаление триггеров

Если нужно остановить автоматическую обработку:

1. Apps Script → левое меню → Triggers (⏰)
2. Найдите триггер
3. Справа нажмите три точки (⋮) → Delete
