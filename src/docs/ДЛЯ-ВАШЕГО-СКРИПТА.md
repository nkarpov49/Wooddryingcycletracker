# 📝 Инструкция для вашего скрипта, который заполняет Google Таблицу

## Краткое описание

Ваш скрипт должен добавлять **одну строку** в Google Таблицу каждый раз, когда завершается цикл сушки.

---

## 📊 Структура строки (6 столбцов)

| Столбец | Что | Тип | Пример | Обязательно |
|---------|-----|-----|--------|-------------|
| **A** | Номер камеры | Число 1-21 | `5` | ✅ Да |
| **B** | Старый номер цикла | Текст (4 цифры) | `0123` | ✅ Да |
| **C** | Новый номер цикла | Текст (4 цифры) | `0124` | ✅ Да |
| **D** | Порода (литовский) | Текст | `Beržas235` | ✅ Да |
| **E** | Дата окончания старого | Дата/время | `2024-02-23 16:00:00` | ✅ Да |
| **F** | Дата начала нового | Дата/время | `2024-02-23 16:05:00` | ✅ Да |

---

## 🎯 Детали по каждому столбцу

### Столбец A: Номер камеры

**Значение:** От 1 до 21

```javascript
// Правильно:
5
12
21

// Неправильно:
0
22
"5"  // не строка, а число!
```

---

### Столбец B: Старый номер цикла (который завершился)

**Формат:** Ровно 4 цифры, с нулями впереди

```javascript
// Правильно:
"0001"
"0123"
"1234"

// Неправильно:
"1"      // нужно 4 цифры
"123"    // нужно 4 цифры
123      // нужна строка, а не число
```

**Как получить:**
- Это `sequentialNumber` из вашей базы данных
- Если номер меньше 4 цифр - добавьте нули впереди
- Если у вас номер `123` → в таблицу пишите `"0123"`

---

### Столбец C: Новый номер цикла (который начинается)

**Формат:** Ровно 4 цифры, с нулями впереди

```javascript
// Правильно:
"0002"  // следующий после 0001
"0124"  // следующий после 0123
"1235"  // следующий после 1234

// Неправильно:
"2"
"124"
124
```

**Логика:**
- Обычно это старый номер + 1
- Но может быть любой следующий свободный номер

---

### Столбец D: Литовское название породы

**⚠️ ВАЖНО:** Используйте **ТОЧНО** эти названия (копируйте отсюда):

```javascript
// Доступные варианты:
"Beržas235"    // Берёза 235
"Berzas235"    // Берёза 235 (без диакритики)
"Beržas285"    // Берёза 285
"Berzas285"    // Берёза 285 (без диакритики)
"Alksnis235"   // Ольха 235
"Ąžuolas235"   // Дуб 235
"Azuolas235"   // Дуб 235 (без диакритики)
"Uosis235"     // Ясень 235
"Klevas235"    // Клён 235
"Skroblas235"  // Скробла 235
```

**Система автоматически конвертирует:**
- `Beržas235` → `Birch235` (в приложении)
- `Klevas235` → `Maple235` (в приложении)
- И так далее

**Проверка:**
- Название должно быть с **ЗАГЛАВНОЙ буквы**
- Число в конце обязательно (235 или 285)
- Если используете другое название - **не будет работать!**

---

### Столбец E: Дата окончания старого цикла

**Формат:** `ГГГГ-ММ-ДД ЧЧ:ММ:СС`

```javascript
// Правильно:
"2024-02-23 16:00:00"
"2024-12-31 23:59:59"
"2023-01-01 08:30:00"

// Или JavaScript Date объект:
new Date()
new Date('2024-02-23T16:00:00')

// Неправильно:
"23.02.2024"
"23-02-2024 16:00"
"2024/02/23"
```

**Когда это происходит:**
- Когда оператор нажимает "Завершить сушку"
- Момент окончания цикла

---

### Столбец F: Дата начала нового цикла

**Формат:** `ГГГГ-ММ-ДД ЧЧ:ММ:СС`

```javascript
// Правильно:
"2024-02-23 16:05:00"  // обычно на несколько минут позже E

// Или:
new Date()

// Неправильно:
"23.02.2024"
```

**Когда это происходит:**
- Обычно сразу после окончания старого
- Или через несколько минут

## 💻 Примеры кода

### Google Apps Script (JavaScript)

```javascript
function addCycleCompletionRow(chamberNum, oldNum, newNum, woodType) {
  // Открываем таблицу
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('Лист1');  // или ваше название
  
  // Текущее время
  const now = new Date();
  const later = new Date(now.getTime() + 5 * 60000); // +5 минут
  
  // Форматируем номера до 4 цифр
  const oldNumFormatted = String(oldNum).padStart(4, '0');
  const newNumFormatted = String(newNum).padStart(4, '0');
  
  // Добавляем строку
  sheet.appendRow([
    chamberNum,           // A: Номер камеры
    oldNumFormatted,      // B: Старый номер (0123)
    newNumFormatted,      // C: Новый номер (0124)
    woodType,             // D: Порода (Beržas235)
    now,                  // E: Конец старого
    later                 // F: Начало нового
  ]);
  
  Logger.log('Добавлена строка для камеры ' + chamberNum);
}

// Использование:
addCycleCompletionRow(5, 123, 124, 'Beržas235');
// Результат:
// 5 | 0123 | 0124 | Beržas235 | 2024-02-23 16:00:00 | 2024-02-23 16:05:00
```

### Python (gspread)

```python
import gspread
from datetime import datetime, timedelta
from oauth2client.service_account import ServiceAccountCredentials

def add_cycle_completion_row(chamber_num, old_num, new_num, wood_type):
    # Подключение к Google Sheets
    scope = ['https://spreadsheets.google.com/feeds',
             'https://www.googleapis.com/auth/drive']
    creds = ServiceAccountCredentials.from_json_keyfile_name('credentials.json', scope)
    client = gspread.authorize(creds)
    
    # Открываем таблицу
    sheet = client.open('Циклы сушки - автосинхронизация').worksheet('Лист1')
    
    # Текущее время
    now = datetime.now()
    later = now + timedelta(minutes=5)
    
    # Форматируем номера до 4 цифр
    old_num_formatted = str(old_num).zfill(4)
    new_num_formatted = str(new_num).zfill(4)
    
    # Форматируем даты
    now_str = now.strftime('%Y-%m-%d %H:%M:%S')
    later_str = later.strftime('%Y-%m-%d %H:%M:%S')
    
    # Добавляем строку
    sheet.append_row([
        chamber_num,           # A: Номер камеры
        old_num_formatted,     # B: Старый номер (0123)
        new_num_formatted,     # C: Новый номер (0124)
        wood_type,             # D: Порода (Beržas235)
        now_str,               # E: Конец старого
        later_str              # F: Начало нового
    ])
    
    print(f'Добавлена строка для камеры {chamber_num}')

# Использование:
add_cycle_completion_row(5, 123, 124, 'Beržas235')
# Результат:
# 5 | 0123 | 0124 | Beržas235 | 2024-02-23 16:00:00 | 2024-02-23 16:05:00
```

## 🔍 Отладка

### Проблема: Старый цикл не закрывается

**Проверьте:**
```javascript
// В вашем коде:
const oldNum = 123;

// В таблице должно быть:
"0123"  // с нулём впереди!

// Не просто:
"123"   // неправильно
```

**Решение:**
```javascript
const oldNumFormatted = String(oldNum).padStart(4, '0');
```

### Проблема: Порода не распознаётся

**Проверьте:**
```javascript
// Правильно:
"Beržas235"  // с заглавной, с числом

// Неправильно:
"beržas235"  // маленькая буква
"Beržas"     // нет числа
"Berzas 235" // есть пробел
```

**Решение:**
Используйте **точные** названия из списка, копируйте их прямо из документации.

### Проблема: Даты неправильные

**Проверьте формат:**
```javascript
// Правильно:
"2024-02-23 16:00:00"

// Неправильно:
"23.02.2024 16:00:00"  // не тот формат
"2024-02-23T16:00:00"  // лишняя T
```

**Решение:**
```javascript
// JavaScript
const date = new Date();
const formatted = date.toISOString().replace('T', ' ').substring(0, 19);
// Результат: "2024-02-23 16:00:00"

// Python
from datetime import datetime
formatted = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
# Результат: "2024-02-23 16:00:00"
```

| Берёза 285 | Birch 285 | `Beržas285` | Birch285 |

| Ясень 235 | Ash 235 | `Uosis235` | Ash235 |
| Клён 235 | Maple 235 | `Klevas235` | Maple235 |
| Скробла 235 | Scroblas 235 | `Skroblas235` | Scroblas235 |

**Пример функции конвертации:**

```javascript
function russianToLithuanian(russianName) {
  const mapping = {
    'Берёза 235': 'Beržas235',
    'Берёза 285': 'Beržas285',
    'Ольха 235': 'Alksnis235',
    'Дуб 235': 'Ąžuolas235',
    'Ясень 235': 'Uosis235',
    'Клён 235': 'Klevas235',
    'Скробла 235': 'Skroblas235'
  };
  return mapping[russianName] || 'Beržas235'; // по умолчанию
}

// Использование:
const woodType = russianToLithuanian('Берёза 235');
// Результат: 'Beržas235'
```

## 🚀 Готово!

Теперь ваш скрипт готов к работе:

1. Когда завершается сушка → ваш скрипт добавляет строку в таблицу
2. Apps Script обнаруживает → отправляет на сервер
3. Сервер обрабатывает → закрывает старый, создаёт новый
4. Оператор видит новый цикл → фотографирует рецепт

**Всё автоматически!** 🎉
