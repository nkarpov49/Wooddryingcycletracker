# Быстрая настройка Google Sheets - Чек-лист

## ✅ Шаг 1: Подготовка Google Таблицы (5 минут)

1. [ ] Создайте новую Google Таблицу
2. [ ] Скопируйте заголовки из файла `/docs/google-sheets-headers.txt` 
3. [ ] Вставьте заголовки в первую строку таблицы
4. [ ] Скопируйте ID таблицы из URL:
   ```
   https://docs.google.com/spreadsheets/d/[ВАШ_ID_ТАБЛИЦЫ]/edit
                                          ^^^^^^^^^^^^^^^^
                                          Скопируйте эту часть
   ```

## ✅ Шаг 2: Настройка Google Cloud (10 минут)

1. [ ] Откройте [Google Cloud Console](https://console.cloud.google.com/)
2. [ ] Создайте новый проект:
   - Нажмите на название проекта вверху
   - "New Project" → Введите название "Drying Chamber App"
   - Нажмите "Create"
   
3. [ ] Включите Google Sheets API:
   - В меню слева: "APIs & Services" → "Library"
   - Найдите "Google Sheets API"
   - Нажмите "Enable"

4. [ ] Создайте Service Account:
   - "APIs & Services" → "Credentials"
   - "Create Credentials" → "Service Account"
   - Имя: "sheets-sync-account"
   - Нажмите "Create and Continue"
   - Role: "Editor" (или "Basic" → "Editor")
   - Нажмите "Done"

5. [ ] Создайте ключ для Service Account:
   - Нажмите на созданный Service Account
   - Вкладка "Keys"
   - "Add Key" → "Create new key"
   - Выберите "JSON"
   - Нажмите "Create" - файл скачается автоматически

6. [ ] Скопируйте email Service Account:
   - В списке Service Accounts скопируйте email (вида `...@...iam.gserviceaccount.com`)

## ✅ Шаг 3: Предоставление доступа к таблице (2 минуты)

1. [ ] Откройте вашу Google Таблицу
2. [ ] Нажмите "Share" (Настройки доступа) в правом верхнем углу
3. [ ] Вставьте email Service Account
4. [ ] Выберите права: "Editor" (Редактор)
5. [ ] Снимите галочку "Notify people" (не нужно отправлять уведомление)
6. [ ] Нажмите "Share"

## ✅ Шаг 4: Подключение в приложении (3 минуты)

1. [ ] Откройте скачанный JSON-файл в текстовом редакторе
2. [ ] Скопируйте ВСЁ содержимое файла (от { до })
3. [ ] В приложении перейдите в режим "Администратор"
4. [ ] Найдите раздел "Настройки синхронизации" или "Google Sheets"
5. [ ] Вставьте JSON в поле "Service Account Credentials"
6. [ ] Вставьте ID таблицы в поле "Spreadsheet ID"
7. [ ] Нажмите "Сохранить" и "Подключить"

## ✅ Шаг 5: Тестирование (5 минут)

1. [ ] Создайте тестовый цикл в приложении:
   - Камера: 1
   - Порядковый номер: TEST-001
   - Код рецепта: TEST
   - Тип дерева: Birch235

2. [ ] Нажмите кнопку "Синхронизировать с Google Sheets"

3. [ ] Проверьте Google Таблицу - должна появиться новая строка

4. [ ] Измените что-нибудь в таблице (например, добавьте комментарий)

5. [ ] Снова синхронизируйте в приложении

6. [ ] Проверьте, что изменения применились в приложении

## ✅ Шаг 6: Импорт существующих данных (опционально)

Если у вас есть старые данные для импорта:

1. [ ] Откройте документацию `/docs/google-sheets-structure.md`
2. [ ] Подготовьте данные в соответствии со структурой
3. [ ] Важно! Для старых циклов:
   - Оставьте `id` пустым - система создаст их
   - Используйте статус `Completed` для завершенных циклов
   - Для фото используйте пустой массив `[]` если фото нет
   
4. [ ] Вставьте данные в таблицу
5. [ ] Запустите синхронизацию
6. [ ] Проверьте корректность импорта в приложении

## 📋 Что заполнять при создании нового цикла в таблице

**Обязательно:**
- `chamberNumber` - номер камеры (1-21)
- `sequentialNumber` - ваш порядковый номер
- `recipeCode` - код рецепта
- `woodType` - один из: Birch235, Birch285, Alder235, Oak235, Ash235, Maple235, Scroblas235
- `status` - всегда `In Progress` для нового цикла
- `createdAt` и `startDate` - текущая дата в формате `2024-02-23T10:00:00Z`

**Остальное:**
- Оставьте пустым или используйте значения по умолчанию
- `id` - ВСЕГДА оставляйте пустым
- `recipePhotos` и `resultPhotos` - используйте `[]`
- `isBaseRecipe` и `isTest` - используйте `FALSE`

## 📋 Что заполнять при завершении цикла

В таблице найдите строку цикла и заполните:
- `status` → измените на `Completed`
- `endDate` → дата окончания в формате `2024-02-25T16:00:00Z`
- `finalMoisture` → конечная влажность (число)
- `qualityRating` → оценка от 0.5 до 5.0
- `overallComment` → ваш комментарий (опционально)

Фотографии результата загружайте через приложение - они автоматически синхронизируются.

## ⚠️ Важные правила

❌ **НЕ ДЕЛАЙТЕ:**
- Не изменяйте `id` после создания записи
- Не заполняйте `recipePhotoUrl` вручную
- Не используйте кириллицу в woodType (только значения из списка)
- Не удаляйте заголовки в первой строке
- Не добавляйте/удаляйте столбцы без обновления кода

✅ **РЕКОМЕНДУЕТСЯ:**
- Делайте резервные копии таблицы регулярно
- Используйте защиту диапазонов для столбца `id`
- Синхронизируйте каждые 5-10 минут (не чаще)
- Проверяйте логи синхронизации при ошибках

## 🆘 Решение проблем

### "Permission denied" при синхронизации
→ Проверьте, что вы дали доступ к таблице для Service Account email

### "Invalid credentials"
→ Проверьте, что JSON скопирован полностью (от { до })

### "Spreadsheet not found"
→ Проверьте правильность ID таблицы

### Записи дублируются
→ Проверьте, что в таблице нет строк с одинаковыми `id`

### Изменения не синхронизируются
→ Проверьте формат дат (должен быть ISO 8601: `YYYY-MM-DDTHH:mm:ssZ`)
