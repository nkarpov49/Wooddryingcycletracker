## ⚡ Самый быстрый путь

Если нужно **ПРЯМО СЕЙЧАС** настроить:

### Шаг 1 (2 мин):
1. Откройте Supabase → скопируйте Project ID и Anon Key
2. Создайте Google Таблицу с 6 столбцами

### Шаг 2 (5 мин):
1. Откройте [`google-apps-script.gs`](google-apps-script.gs)
2. Скопируйте весь код
3. В Google Таблице: Расширения → Apps Script
4. Вставьте код
5. Измените 3 константы: URL_WEBHOOK, AUTH_TOKEN, SHEET_NAME

### Шаг 3 (3 мин):
1. Запустите функцию `testConfiguration()`
2. Дайте разрешения
3. Добавьте тестовую строку в таблицу
4. Запустите `manualSync()`
5. Подождите 3 минуты
6. Проверьте приложение

### Шаг 4 (1 мин):
1. Запустите `setupTrigger()` или `setupTimeTrigger()`
2. Готово!

## 🔍 Что нужно знать про вашу систему

### Ваш скрипт должен добавлять в Google Таблицу строку с 6 столбцами:

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 5 | 0123 | 0124 | Beržas235 | 2024-02-23 16:00 | 2024-02-23 16:05 |

Где:
- **A** = Номер камеры (1-21)
- **B** = Старый номер цикла (4 цифры: `0123`)
- **C** = Новый номер цикла (4 цифры: `0124`)
- **D** = Литовское название породы (`Beržas235`, `Klevas235`, и т.д.)
- **E** = Дата окончания старого цикла
- **F** = Дата начала нового цикла

**Важно:**
- Номера циклов - обязательно строки и 4 цифры с нулями: `"0001"`, `"0123"`
- Породы - точные литовские названия (они автоматически конвертируются в английские)
- Даты - формат `ГГГГ-ММ-ДД ЧЧ:ММ:СС`

Подробнее: [`ДЛЯ-ВАШЕГО-СКРИПТА.md`](ДЛЯ-ВАШЕГО-СКРИПТА.md)

## 🎯 Литовские названия пород (для вашего скрипта)

Используйте **ТОЧНО** эти названия в столбце D:

```
Beržas235   (или Berzas235)   → конвертируется в Birch235 (Берёза 235)
Beržas285   (или Berzas285)   → конвертируется в Birch285 (Берёза 285)
Alksnis235                    → конвертируется в Alder235 (Ольха 235)
Ąžuolas235  (или Azuolas235)  → конвертируется в Oak235 (Дуб 235)
Uosis235                      → конвертируется в Ash235 (Ясень 235)
Klevas235                     → конвертируется в Maple235 (Клён 235)
Skroblas235                   → конвертируется в Scroblas235 (Скробла 235)
```

Система автоматически конвертирует литовские названия в английские для приложения!

- ✅ Работает дальше

## 🆘 Если что-то не работает

### Проверьте в таком порядке:

1. **Apps Script логи** (Вид → Журналы)
   - Есть ошибки?
   - Видите "✅ Успех!"?

2. **Константы в коде**
   - `URL_WEBHOOK` - правильный Project ID?
   - `AUTH_TOKEN` - правильный Anon Key?
   - `SHEET_NAME` - точное название листа?

3. **Данные в таблице**
   - 6 столбцов заполнены?
   - Номера - строки из 4 цифр?
   - Порода - из списка литовских?
   - Даты - правильный формат?

4. **Приложение**
   - Существует ли старый цикл?
   - Его sequentialNumber = старому номеру?

5. **Триггеры**
   - Apps Script → ⏰ Triggers
   - Есть триггер в списке?

## 📞 Нужна помощь?

### Частые вопросы:

**Q: Можно ли создавать циклы вручную в приложении?**  
A: Нет, все циклы создаются автоматически через Google Sheets.

**Q: Что делает оператор в приложении?**  
A: Только фотографирует рецепт. Создание циклов - автоматическое.

**Q: Что делает лидер?**  
A: Завершает циклы, добавляет фото результата, оценку, комментарий.

**Q: Можно ли массово добавлять строки?**  
A: Да, но лучше с паузой 10-15 секунд между строками.

**Q: Старый цикл не закрывается?**  
A: Проверьте, что номер в таблице точно совпадает с sequentialNumber в приложении (с нулями!)

**Q: Порода не конвертируется?**  
A: Используйте точные литовские названия из списка выше.
