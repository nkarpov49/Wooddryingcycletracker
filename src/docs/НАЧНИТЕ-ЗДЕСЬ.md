## 🚨 Частые проблемы и решения

### ❌ "Строка уже обработана"

➡️ Откройте **[РЕШЕНИЕ-ПРОБЛЕМЫ-ТЕСТИРОВАНИЯ.md](./РЕШЕНИЕ-ПРОБЛЕМЫ-ТЕСТИРОВАНИЯ.md)**

**Быстрое решение:**
1. В Apps Script запустите `resetRowStatus()`
2. Затем запустите `manualSync()`

---

### ❌ "Не могу найти цикл в базе"

➡️ Откройте **[КАК-НАЙТИ-ЦИКЛЫ-В-БАЗЕ.md](./КАК-НАЙТИ-ЦИКЛЫ-В-БАЗЕ.md)**

**Быстрое решение:**
1. Откройте **[ПРОВЕРКА-ЦИКЛОВ-В-БАЗЕ.html](./ПРОВЕРКА-ЦИКЛОВ-В-БАЗЕ.html)** в браузере
2. Нажмите "Загрузить циклы"
3. Увидите все циклы из базы

---

### ❌ "Ошибка 401 Unauthorized"

**Причина:** Неправильный токен

**Решение:**
1. Откройте Google Sheets
2. Apps Script → Файл → Свойства проекта
3. Проверьте `AUTH_TOKEN` в коде скрипта
4. Должен совпадать с `SUPABASE_ANON_KEY`

---

### ❌ "Порода не конвертируется"

**Причина:** Неизвестное литовское название

**Решение:**
Добавьте маппинг в `/supabase/functions/server/wood-type-mapping.ts`:
```typescript
const lithuanianToEnglishMap: Record<string, string> = {
  'Beržas235': 'Birch235',
  'НовоеНазвание235': 'NewWood235',  // ← Добавьте сюда
  ...
};
```

---

### ❌ "Старый цикл не закрывается"

**Причины:**
1. Номер в таблице не совпадает с sequentialNumber в базе
2. Камера не совпадает
3. Цикла вообще нет в базе

**Решение:**
1. Используйте **[ПРОВЕРКА-ЦИКЛОВ-В-БАЗЕ.html](./ПРОВЕРКА-ЦИКЛОВ-В-БАЗЕ.html)**
2. Найдите правильный sequentialNumber
3. Используйте ТОЧНО этот номер в таблице

## 📊 Инструменты отладки

### 1. HTML просмотр базы

**Файл:** `ПРОВЕРКА-ЦИКЛОВ-В-БАЗЕ.html`

**Возможности:**
- Просмотр всех циклов
- Поиск по номеру/камере

В Google Sheets:
```
Расширения → Apps Script → Выполнить функцию → Вид → Журналы
```

### 3. Supabase логи

В Supabase Dashboard:
```
Edge Functions → make-server-c5bcdb1f → Logs
```

**Логи синхронизации:**
```
GET /sheets/sync-logs?limit=10
```

