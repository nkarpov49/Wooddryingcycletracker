# ✅ Правильная схема импорта из Google Sheets

## 🎯 Главный принцип

**Из Google Sheets импортируются 7 полей:**
1. **Номер камеры** (1-21) - ОДИН И ТОТ ЖЕ для старого и нового
2. **Порода дерева** для НОВОГО цикла (литовское название)
3. **Старый порядковый номер** (например: 9954)
4. **Дата окончания СТАРОГО** цикла
5. **Новый порядковый номер** (например: 9955)
6. **Дата начала НОВОГО** цикла

**Логика:**
- **Старый цикл:** Порода УЖЕ ЕСТЬ в карточке → только закрываем (добавляем endDate)
- **Новый цикл:** Порода БЕРЁТСЯ из таблицы → конвертируется → создаём с породой

---

## 📊 Структура Google Sheets

```
| A      | B           | C      | D           | E      | F           |
|--------|-------------|--------|-------------|--------|-------------|
| Камера | Порода      | Старый | Дата оконч. | Новый  | Дата начала |
|        | (новый цикл)| №      | (старый)    | №      | (новый)     |
|--------|-------------|--------|-------------|--------|-------------|
| 18     | Beržas235   | 9954   | 2026-02-23  | 9955   | 2026-02-23  |
| 5      | Alksnis235  | 9321   | 2026-02-24  | 9322   | 2026-02-24  |
| 12     | Ąžuolas235  | 8756   | 2026-02-25  | 8757   | 2026-02-25  |
```

**Пояснение столбцов:**
- **A (Камера):** Номер камеры 1-21 (ОДИН И ТОТ ЖЕ для старого и нового цикла)
- **B (Порода):** Литовское название породы для НОВОГО цикла (будет конвертировано в английский)
- **C (Старый №):** 4-значный порядковый номер СТАРОГО цикла который нужно закрыть
- **D (Дата оконч.):** Дата окончания СТАРОГО цикла
- **E (Новый №):** 4-значный порядковый номер НОВОГО цикла (обычно старый + 1)
- **F (Дата начала):** Дата начала НОВОГО цикла (обычно = дата окончания старого)

## 🔄 Как это работает

### Пример: Камера 18

#### Исходное состояние (в базе данных):
```javascript
{
  id: "abc-123",
  chamberNumber: 18,
  sequentialNumber: "9954",
  woodType: "Birch285",  // ← Лидер выбрал в начале цикла
  status: "In Progress",
  startDate: "2026-02-20T08:00:00Z",
  endDate: null,  // ← Пока не завершён
  recipePhotos: [...]  // ← Лидер загрузил фото рецепта
}
```

#### Шаг 1: Оператор добавляет строку в Google Sheets

```
18 | Beržas235 | 9954 | 2026-02-23 | 9955 | 2026-02-23
```

#### Шаг 2: Apps Script отправляет на сервер

```json
{
  "rowNumber": 2,
  "chamberNumber": 18,
  "woodTypeLithuanian": "Beržas235",
  "oldSequentialNumber": "9954",
  "oldCycleEndDate": "2026-02-23T10:00:00Z",
  "newSequentialNumber": "9955",
  "newCycleStartDate": "2026-02-23T10:03:00Z"
}
```

#### Шаг 3: Сервер закрывает старый цикл

```javascript
// Находит цикл: chamberNumber=18 + sequentialNumber="9954"
// Обновляет ТОЛЬКО 2 поля:
{
  id: "abc-123",  // ← Остаётся прежним
  chamberNumber: 18,  // ← Остаётся
  sequentialNumber: "9954",  // ← Остаётся
  woodType: "Birch285",  // ← Остаётся (была выбрана Лидером)
  status: "Completed",  // ← ОБНОВЛЕНО
  startDate: "2026-02-20T08:00:00Z",  // ← Остаётся
  endDate: "2026-02-23T10:00:00Z",  // ← ОБНОВЛЕНО
  recipePhotos: [...]  // ← Остаются
  // Все остальные поля тоже остаются!
}
```

#### Шаг 4: Сервер ждёт 2.5 минуты

```
⏳ Задержка для избежания конфликтов...
```

#### Шаг 5: Сервер создаёт новый цикл

```javascript
// Конвертирует: "Beržas235" → "Birch235"
// Создаёт НОВЫЙ цикл с данными из таблицы:
{
  id: "xyz-789",  // ← Новый UUID
  chamberNumber: 18,  // ← Из таблицы (тот же)
  sequentialNumber: "9955",  // ← Из таблицы (+1)
  woodType: "Birch235",  // ← Из таблицы (конвертировано)
  recipeCode: "Birch235-9955",  // ← Сгенерирован
  status: "In Progress",
  startDate: "2026-02-23T10:03:00Z",  // ← Из таблицы
  endDate: null,
  recipePhotos: [],  // ← Пусто, Лидер загрузит позже
  finalMoisture: undefined,
  qualityRating: undefined,
  // ... остальные поля пустые
}
```

#### Результат:

**Старый цикл (завершён):**
- UUID: abc-123
- Камера: 18
- Номер: 9954
- Порода: Birch285 (осталась как была)
- Статус: Completed ✅
- Дата окончания: 2026-02-23

**Новый цикл (активный):**
- UUID: xyz-789
- Камера: 18
- Номер: 9955
- Порода: Birch235 (из таблицы)
- Статус: In Progress ⚙️
- Дата начала: 2026-02-23

## 📝 Полный workflow

### День 1 (20 февраля, 08:00)
**Лидер в приложении:**
- Видит новый цикл 18/9954
- Выбирает породу: Birch285
- Загружает фото рецепта
- Цикл начинается

### День 4 (23 февраля, 10:00)
**Оператор завершает сушку:**
- Добавляет строку в Google Sheets:
  ```
  18 | Beržas235 | 9954 | 2026-02-23 | 9955 | 2026-02-23
  ```
  *(Для нового цикла решили использовать Beržas235 вместо Beržas285)*

**Google Apps Script:**
- Видит новую строку
- Отправляет данные на сервер

**Сервер (автоматически):**
- Закрывает цикл 18/9954 (endDate = 2026-02-23)
- Ждёт 2.5 минуты
- Создаёт цикл 18/9955 с породой Birch235 (конвертирована из Beržas235)

### День 4 (23 февраля, 10:05)
**Оператор в приложении:**
- Обновляет список
- Видит камеру 18 (новый цикл 9955, порода: Birch235)
- Может сфотографировать рецепт если нужно

**Лидер в приложении:**
- Видит завершённый цикл 18/9954 (Birch285)
- Добавляет влажность, рейтинг, фото результатов
- Видит новый цикл 18/9955 (Birch235)
- Загружает фото рецепта для нового цикла

## 🧪 Как протестировать

### Шаг 1: Подготовьте тестовые данные

1. Откройте `/docs/check-cycles-database.html`
2. Найдите любой цикл "В процессе"
3. Запишите:
   ```
   Камера: 18
   Номер: 9954
   Порода: Birch285 (например)
   ```

### Шаг 2: Создайте строку в Google Sheets

Структура таблицы (заголовки в строке 1):
```
| A      | B          | C    | D          | E    | F          |
| Камера | Порода     | №    | Дата оконч | №    | Дата нач.  |
```

Добавьте данные в строку 2:
```
| 18 | Beržas235 | 9954 | 2026-02-23 | 9955 | 2026-02-23 |
```

**Важно:**
- Камера: 18 (номер тестового цикла)
- Порода: Beržas235 (литовское название для НОВОГО цикла)
- Старый №: 9954 (из HTML инструмента)
- Дата оконч: 2026-02-23 (сегодня или позже)
- Новый №: 9955 (9954 + 1)
- Дата нач: 2026-02-23 (обычно = дата окончания)

### Шаг 3: Обновите Apps Script

1. Google Sheets → Расширения → Apps Script
2. Вставьте код из `/docs/google-apps-script.gs`
3. Сохраните (Ctrl+S)

## 🎉 Готово!

Теперь система работает правильно:
1. **Оператор** добавляет строку с 6 полями в Google Sheets
