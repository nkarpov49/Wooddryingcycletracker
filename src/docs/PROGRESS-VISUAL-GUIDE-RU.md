## 📐 Технические параметры

### SVG Circle (прогресс-бар)
```javascript
radius: 45
circumference: 2 × π × 45 = 282.7
strokeWidth: 8
viewBox: "0 0 100 100"

// Расчёт offset для прогресса
offset = circumference - (progress / 100) × circumference
```

### Динамический размер шрифта процента
```javascript
if (progressStr.length >= 5) return 'text-lg';  // 115.5% (5 символов)
```

## 🎯 Логика отображения

### Проценты >100%
```javascript
// Для круга используем modulo 100
displayProgress = progress > 100 ? progress % 100 : progress;

// Примеры:
115.5% → круг заполнен на 15.5%
103.2% → круг заполнен на 3.2%
99.8%  → круг заполнен на 99.8%
```

### Критическая зона
```javascript
isCritical = (
  progress >= 99 && 
  progress <= 103 && 
  !hasRecipePhoto
);
```

### Выбор цвета
```javascript
function getProgressColor() {
  if (isCritical) return 'stroke-red-500';
  if (progress >= 100) return 'stroke-purple-500';  // >100%
  if (progress >= 90) return 'stroke-orange-500';
  if (progress >= 70) return 'stroke-yellow-500';
  if (progress >= 40) return 'stroke-blue-500';
  return 'stroke-green-500';
}
```
