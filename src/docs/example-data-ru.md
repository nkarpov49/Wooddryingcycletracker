# Примеры данных для Google Таблицы

## Пример 1: Новый цикл (только начался)

```
[пусто]  5  2024-045  БР235-48-30  Birch235  In Progress  2024-02-23T09:00:00Z  2024-02-23T09:00:00Z  [пусто]  FALSE  FALSE  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  [пусто]  []  []  [пусто]  [пусто]
```

**Расшифровка:**
- **id:** пусто (сгенерируется автоматически)
- **chamberNumber:** 5
- **sequentialNumber:** 2024-045
- **recipeCode:** БР235-48-30
- **woodType:** Birch235
- **status:** In Progress
- **createdAt:** 2024-02-23T09:00:00Z
- **startDate:** 2024-02-23T09:00:00Z
- Остальное пусто или FALSE/[]

## Таблица быстрого копирования

### Заголовки (1-я строка):
```
id	chamberNumber	sequentialNumber	recipeCode	woodType	status	createdAt	startDate	customWoodType	isBaseRecipe	isTest	endDate	finalMoisture	qualityRating	overallComment	startTemperature	startWeatherCode	avgTemp	avgDayTemp	avgNightTemp	minTemp	maxTemp	recipePhotos	resultPhotos	recipePhotoPath	recipePhotoUrl
```

### Шаблон новой строки (2-я строка и далее):
```
	[НОМЕР_КАМЕРЫ]	[ВАШ_НОМЕР]	[КОД_РЕЦЕПТА]	Birch235	In Progress	2024-02-23T10:00:00Z	2024-02-23T10:00:00Z		FALSE	FALSE							[]	[]		
```

**Что нужно заменить:**
- `[НОМЕР_КАМЕРЫ]` → число от 1 до 21
- `[ВАШ_НОМЕР]` → ваш порядковый номер (например: 2024-001)
- `[КОД_РЕЦЕПТА]` → код рецепта (например: БР235-48-30)
- Дату можно обновить на текущую

## Быстрая генерация даты в формате ISO

### Для Google Sheets формула:
```
=TEXT(NOW(),"YYYY-MM-DD")&"T"&TEXT(NOW(),"HH:mm:ss")&"Z"
```

### Для Excel формула:
```
=TEXT(NOW(),"yyyy-mm-dd")&"T"&TEXT(NOW(),"hh:mm:ss")&"Z"
```

### Вручную:
```
Формат: YYYY-MM-DDTHH:mm:ssZ
Пример: 2024-02-23T14:30:00Z

Где:
- YYYY = год (2024)
- MM = месяц (02)
- DD = день (23)
- T = разделитель (всегда T)
- HH = часы (14)
- mm = минуты (30)
- ss = секунды (00)
- Z = UTC timezone (всегда Z)
```

## Оценки качества (qualityRating)

Используйте следующие значения:
```
0.5 - Очень плохо (критические дефекты)
1.0 - Плохо
1.5 - Неудовлетворительно
2.0 - Ниже среднего
2.5 - Почти средне
3.0 - Средне (🔴 красная карточка в приложении)
3.5 - Выше среднего
4.0 - Хорошо
4.5 - Очень хорошо
5.0 - Отлично (идеальное качество)
```

**Важно:** Рецепты с рейтингом < 3.0 будут отображаться с красным фоном в приложении!

## JSON формат для фотографий

### Пустой массив (нет фото):
```json
[]
```

### Одна фотография:
```json
[{"path":"photo_001.jpg","caption":"Описание"}]
```

### Несколько фотографий:
```json
[{"path":"photo_001.jpg","caption":"Вид спереди"},{"path":"photo_002.jpg","caption":"Вид сбоку"}]
```

**Важно:** 
- Используйте двойные кавычки `"`
- НЕ добавляйте пробелы после запятых (или Google Sheets может неправильно распарсить)
- Поле `url` не добавляйте - оно генерируется автоматически

## Миграция старых данных

Если у вас есть старые записи БЕЗ некоторых полей:

### Минимально необходимые поля:
```
id: [пусто]
chamberNumber: [число 1-21]
sequentialNumber: [ваш номер]
recipeCode: [код]
woodType: [тип из списка]
status: Completed
createdAt: [дата]
startDate: [дата]
```

### Остальное можно оставить пустым:
- Приложение автоматически установит `status = "Completed"` если есть finalMoisture + qualityRating + resultPhotos
- Погодные данные можно не заполнять для старых записей
- Фотографии можно добавить позже через приложение

### Пример старой записи:
```
[пусто]	8	OLD-123	СТР-РЕЦ	Birch235	Completed	2023-12-01T10:00:00Z	2023-12-01T10:00:00Z		TRUE	FALSE	2023-12-05T16:00:00Z	8.0	4.0	Хороший результат						[]	[]		
```

Такая запись будет корректно импортирована, и вы сможете добавить фотографии позже.

## Массовое создание записей

Если нужно создать много циклов сразу:

1. Скопируйте шаблон строки
2. Вставьте нужное количество раз
3. Измените только:
   - chamberNumber (номер камеры)
   - sequentialNumber (уникальный для каждой)
   - recipeCode (если разные)
   - woodType (если разные)
4. Остальное можно оставить одинаковым
5. После синхронизации каждая строка получит уникальный `id`
