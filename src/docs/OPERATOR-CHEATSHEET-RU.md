# 📱 Шпаргалка для операторов: Проценты завершения

## 🎨 Что означают цвета круга?

```
    🟢 Зелёный        0-39%     Только начали
    
    🔵 Синий         40-69%     Идёт процесс
    
    🟡 Жёлтый        70-89%     Близко к концу
    
    🟠 Оранжевый     90-98%     Почти готово
    
    🔴 Красный       99-103%    ⚠️ СФОТОГРАФИРУЙТЕ!
    (мигает)
    
    🟣 Фиолетовый    >103%      Превышение нормы
```

---

## 🔴 Что делать при красном мигании?

### ⚠️ Карточка камеры мигает красным

**Это значит:**
- Процент достиг 99-103%
- Фото рецепта **не загружено**
- **Нужно срочно сфотографировать!**

**Действия:**
1. 📱 Нажмите на мигающую карточку камеры
2. 📸 Сфотографируйте рецепт
3. ✅ Нажмите "Готово"
4. 🟢 Мигание прекратится автоматически

---

## 🔔 Уведомления

### Когда приходят?
- ⏰ Каждые **2 минуты**
- 🔴 Пока процент 99-103%
- 📸 Пока **нет фото** рецепта

### Как выглядят?
```
╔═══════════════════════════════════╗
║  🔔 Kiln Drying Tracker           ║
║                                   ║
║  ⚠ Камера 7: 101% - Требуется    ║
║    фото!                          ║
║                                   ║
║  Пожалуйста, сфотографируйте      ║
║  рецепт                           ║
╚═══════════════════════════════════╝
```

### Звук
```
♪ Beep! Beep! Beep! (3 коротких сигнала)
```

### Как остановить?
✅ **Загрузите фото рецепта** → Уведомления прекратятся сами

## 📸 Статус фото

### ✓ Фото (зелёный)
```
✅ Фото рецепта загружено
✅ Всё в порядке
```

### ✗ Нет фото (красный)
```
❌ Фото не загружено
⚠️ При 99-103% карточка мигает!
```

## 🎯 Краткая инструкция

### Нормальная работа
1. 👀 Смотрите на цвет круга
2. 🟢🔵🟡 Зелёный/Синий/Жёлтый = Всё идёт
3. 🟠 Оранжевый = Скоро готово
4. 🔴 Красный (мигает) = **ДЕЙСТВУЙТЕ!**

### Критическая ситуация (99-103%)
1. 🔴 Карточка мигает красным
2. 🔔 Приходит уведомление
3. 📸 **Сфотографируйте рецепт**
4. ✅ Мигание прекращается

### Превышение нормы (>103%)
1. 🟣 Фиолетовый круг
2. 📸 Сфотографируйте рецепт (если не сделали)
3. ✅ Цикл можно завершать

## 🆘 Частые вопросы

**❓ Карточка мигает красным, что делать?**  
📸 Сфотографируйте рецепт! Это означает процент 99-103% без фото.

**❓ Уведомления приходят постоянно**  
📸 Загрузите фото рецепта, они прекратятся автоматически.

**❓ Процент >100%, это нормально?**  
✅ Да! Некоторые рецепты могут превышать 100%. Фиолетовый цвет.

**❓ Круг почти пустой при 115%**  
✅ Это нормально! Круг показывает остаток (115% = 15% круга).

**❓ Проценты не обновляются**  
🔄 Обновляются каждые 15 минут. Подождите немного.

**❓ Не слышу звук уведомлений**  
🔊 Проверьте громкость браузера и системы.
