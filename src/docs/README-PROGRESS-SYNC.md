# 📊 Синхронизация процентов завершения циклов сушки

## 🎯 Что это?

Автоматическая синхронизация **процентов завершения** из Google Sheets в интерфейс оператора каждые 15 минут.

## ✨ Возможности

### Визуализация
- 🎨 **Циркулярные прогресс-бары** (дизайн Apple Watch)
- 🌈 **Цветовое кодирование** по стадиям (зелёный → синий → жёлтый → оранжевый → красный → фиолетовый)
- 📈 **Поддержка трёхзначных процентов** с десятичными (115.5%)
- 🔴 **Мигающие красные карточки** при критическом проценте (99-103%) без фото

### Уведомления
- 🔔 **Push-уведомления** браузера
- 🔊 **Звуковые сигналы** (3 beep-а)
- ⏰ **Автоповтор каждые 2 минуты** до загрузки фото
- ✅ **Автоотключение** после сохранения рецепта

### Интеграция
- ⚡ **Синхронизация каждые 15 минут** из Google Sheets
- 🔄 **Автоматическое обновление** активных циклов
- 📡 **Двусторонняя связь** Sheets ↔ Make

## 🚀 Быстрый старт (5 минут)

### 1. Откройте Google Apps Script
```
Google Sheets → Расширения → Apps Script
```

### 2. Скопируйте код
Откройте файл: `/docs/google-apps-script-progress.txt`

### 3. Замените URL
```javascript
const SUPABASE_URL = 'https://ваш-project-id.supabase.co';
```

### 4. Создайте триггер
```
Triggers (часы) → Add Trigger
Function: syncProgressToApp
Event: Time-driven → Minutes → Every 15 minutes
```

### 5. Запустите тест
```
Run ▶️ → Разрешите доступ → Проверьте логи
```

**Готово!** ✅

### Формула процента
```
=(now()-$C$6)/((vlookup($C$4;Standartai!$B$5:$E;4;false))/24)
```
Возвращает дробное число (0.592 = 59.2%), автоматически умножается на 100.

### Цветовая схема
| Процент | Цвет | Статус |
|---------|------|--------|
| 0-39% | 🟢 Зелёный | Начало |
| 40-69% | 🔵 Синий | Середина |
| 70-89% | 🟡 Жёлтый | Подготовка |
| 90-98% | 🟠 Оранжевый | Почти готово |
| 99-103% | 🔴 Красный | **КРИТИЧНО! Требуется фото** |
| >103% | 🟣 Фиолетовый | Превышение |

## 🐛 Частые проблемы

### ❌ Проценты неправильные (2, 3, 1)
**Причина**: Старая версия кода  
**Решение**: Скопируйте обновлённый код из `google-apps-script-progress.txt`

### ❌ Ошибка "urlfetch limit"
**Причина**: Слишком много тестовых запусков или дублирующиеся триггеры  
**Решение**: Удалите лишние триггеры, подождите до завтра

### ❌ Уведомления не работают
**Причина**: Не разрешены в браузере или камера не в критической зоне  
**Решение**: Разрешите уведомления, проверьте процент 99-103% без фото

### ❌ Процент >100% но круг пустой
**Причина**: Это нормально! Круг показывает остаток (modulo 100)  
**Пример**: 115.5% → круг заполнен на 15.5% (фиолетовый)

## 🎯 Для операторов

### Когда приходят уведомления?
- Процент достиг 99-103%
- Фото рецепта **не загружено**
- Повторяются каждые 2 минуты

### Как остановить уведомления?
- Сфотографируйте рецепт
- Уведомления прекратятся автоматически

### Что означают цвета?
- 🟢 **Зелёный**: Только начали (0-39%)
- 🔵 **Синий**: Идёт процесс (40-69%)
- 🟡 **Жёлтый**: Близко к концу (70-89%)
- 🟠 **Оранжевый**: Почти готово (90-98%)
- 🔴 **Красный (мигает)**: **Срочно сфотографируйте!** (99-103%)
- 🟣 **Фиолетовый**: Превышение нормы (>103%)
