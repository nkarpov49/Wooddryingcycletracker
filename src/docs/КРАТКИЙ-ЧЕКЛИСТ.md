- [ ] Сохранил куда-то оба значения

```
Anon Key:   _______________________
```

## ЧАСТЬ 6: Ваш скрипт (настройте под себя)

### Что должен делать ваш скрипт:

Добавлять строку в таблицу с 6 столбцами:

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 5 | 0123 | 0124 | Beržas235 | 2024-02-23 16:00:00 | 2024-02-23 16:05:00 |

### Требования:

- [ ] Столбец A: число 1-21
- [ ] Столбец B: строка 4 цифры (с нулями: `"0123"`)
- [ ] Столбец C: строка 4 цифры (с нулями: `"0124"`)
- [ ] Столбец D: литовское название из списка
- [ ] Столбец E: дата формат `ГГГГ-ММ-ДД ЧЧ:ММ:СС`
- [ ] Столбец F: дата формат `ГГГГ-ММ-ДД ЧЧ:ММ:СС`

### Литовские названия пород:

```
Beržas235  или  Berzas235   (Берёза 235)
Beržas285  или  Berzas285   (Берёза 285)
Alksnis235                  (Ольха 235)
Ąžuolas235 или  Azuolas235  (Дуб 235)
Uosis235                    (Ясень 235)
Klevas235                   (Клён 235)
Skroblas235                 (Скробла 235)
```

### Примеры кода:

**JavaScript (Google Apps Script):**
```javascript
sheet.appendRow([
  5,                          // A: камера
  '0123',                     // B: старый номер
  '0124',                     // C: новый номер
  'Beržas235',               // D: порода
  '2024-02-23 16:00:00',     // E: конец старого
  '2024-02-23 16:05:00'      // F: начало нового
]);
```

**Python (gspread):**
```python
worksheet.append_row([
    5,                          # A: камера
    '0123',                     # B: старый номер
    '0124',                     # C: новый номер
    'Beržas235',               # D: порода
    '2024-02-23 16:00:00',     # E: конец старого
    '2024-02-23 16:05:00'      # F: начало нового
])
```

- [ ] Адаптировал код под мою систему
- [ ] Протестировал добавление строки

## ЧАСТЬ 7: Проверка работы (после настройки вашего скрипта)

### Когда ваш скрипт добавит строку:

**Через 10 секунд:**
- [ ] Старый цикл закрыт в приложении

**Через 3 минуты:**
- [ ] Новый цикл создан в приложении
- [ ] Порода конвертирована (литовский → английский)

### Логи:

**Apps Script:**
- [ ] Расширения → Apps Script → Вид → Журналы выполнения
- [ ] Вижу: "✅ Успех! Строка X отправлена"

**Сервер:**
```
GET https://МОЙ_ID.supabase.co/functions/v1/make-server-c5bcdb1f/sheets/sync-logs?limit=10
Authorization: Bearer МОЙ_ANON_KEY
```
- [ ] Вижу JSON с логами синхронизации

## 🆘 Если не работает

### 1. Проверьте Apps Script логи:
- Вид → Журналы выполнения
- Есть ошибки?

### 2. Проверьте константы:
- `URL_WEBHOOK` - правильный Project ID?
- `AUTH_TOKEN` - правильный Anon Key?
- `SHEET_NAME` - точное название листа?

### 3. Проверьте данные в таблице:
- Номер камеры 1-21?
- Номера циклов 4 цифры со строкой?
- Порода из списка литовских?
- Даты в правильном формате?

### 4. Проверьте приложение:
- Существует ли старый цикл с таким номером?
- Есть ли у него поле sequentialNumber = старому номеру?

### 5. Проверьте триггеры:
- Apps Script → иконка ⏰ → есть триггер в списке?
