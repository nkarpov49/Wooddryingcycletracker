# 🚀 Быстрый старт - 5 минут

## Что вам нужно знать

### ✅ Обязательные столбцы для НОВОГО цикла:

Скопируйте эту строку в Google Sheets (после заголовков):

```
	[КАМЕРА]	[ВАШ-НОМЕР]	[КОД]	Birch235	In Progress	2024-02-23T10:00:00Z	2024-02-23T10:00:00Z		FALSE	FALSE							[]	[]		
```

**Замените:**
- `[КАМЕРА]` → число 1-21
- `[ВАШ-НОМЕР]` → например: 2024-001
- `[КОД]` → код рецепта, например: БР235-48-30

**Оставьте как есть всё остальное!**

---

## ✅ Обязательные столбцы для ЗАВЕРШЕНИЯ цикла:

В существующей строке измените:

```
status:         Completed
endDate:        2024-02-25T16:00:00Z
finalMoisture:  8.2
qualityRating:  4.5
```

---

## 📋 Типы древесины (выберите один):

- `Birch235` - Береза 235
- `Birch285` - Береза 285
- `Alder235` - Ольха 235
- `Oak235` - Дуб 235
- `Ash235` - Ясень 235
- `Maple235` - Клен 235
- `Scroblas235` - Скробла 235

---

## ⭐ Оценка качества:

- `5.0` = Отлично
- `4.5` = Очень хорошо
- `4.0` = Хорошо
- `3.5` = Выше среднего
- `3.0` = Средне (⚠️ карточка станет красной!)
- `2.5` и ниже = Плохо (🔴 красная карточка)

---

## ❌ Что НЕ трогать:

- **id** - всегда пусто для новых, не меняйте для существующих
- **recipePhotoUrl** - генерируется автоматически
- **startTemperature, avgTemp** и т.д. - погода добавится автоматически

---

## 📸 Фотографии:

**Не загружайте фото в Google Sheets!**

Используйте приложение:
1. Оператор загружает фото рецепта через кнопку "Сфотографировать"
2. Лидер загружает фото результата при завершении
3. Пути к фото автоматически синхронизируются в таблицу

---

## 🔄 Синхронизация:

**Автоматически каждые 5 минут**

Или вручную: Администратор → Кнопка "Синхронизировать"

---

## 🎯 Три главных правила:

1. **Не меняйте `id` после создания**
2. **Используйте только типы дерева из списка**
3. **Фотографии только через приложение**

1. Откройте `/docs/google-sheets-headers.txt`
