# 📄 Шпаргалка Google Sheets - Распечатайте и повесьте на стену!

---

## ✅ НОВЫЙ ЦИКЛ - Копируй и вставляй

```
	[1-21]	[НОМЕР]	[КОД]	Birch235	In Progress	2024-02-23T10:00:00Z	2024-02-23T10:00:00Z		FALSE	FALSE							[]	[]		
```

**Замени 3 значения:**
- `[1-21]` = номер камеры
- `[НОМЕР]` = твой номер (2024-001)
- `[КОД]` = код рецепта (БР235-48-30)


## 🆘 ЧТО-ТО СЛОМАЛОСЬ?

1. ❓ Не синхронизируется?
   → Проверь формат даты (должна быть T и Z)

2. ❓ Запись пропала?
   → Не меняй `id` после создания!

3. ❓ Красная карточка?
   → Оценка меньше 3.0

4. ❓ Фото не видно?
   → Загружай через приложение, не Google Sheets!

7. Удали тестовую строку из таблицы
